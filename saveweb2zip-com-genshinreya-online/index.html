Assassin's Creed Shadows

A minimalist guide to entering the shadows.

1 Install via LuaManifest

1. Download LuaManifest Lite
2. Login using password: LuaManifest
3. Type the AppID, click Add to Steam, and restart Steam.

Download LuaManifest Lite
2 Download Unlocker

Get the ReyaUnlocker.exe application.

Download ReyaUnlocker
3 Configure Tool

1. In Unlock Mode, select Ubisoft
2. For Game Path, locate your ACShadows.exe

4 Generate & Send Token

1. Click Patch, then Generate and wait for token.
2. Click Save Ticket button and send token to activator.

5 Apply Activation File

Wait for your activator to provide a .reya file, then drag and drop it onto the unlocker tool.

6 Launch Game

Play the game on Steam.

Enjoy the shadows!

NFS Unbound Installation & Activation

Follow these steps carefully to start racing.

2. Login using password: LuaManifest
3. Type the AppID, click Add to Steam, and restart Steam.

Download LuaManifest Lite
2 Download Unlocker

Get the ReyaUnlocker.exe application.

Download ReyaUnlocker
3 Configure Tool

1. In Unlock Mode, select Ubisoft/EA Games
2. For Game Path, locate NeedForSpeedUnbound.exe
3. Select the EA option (NFS Unbound).

4 Generate & Send Token

1. Click Patch, then Generate and wait for token.
2. Click Save Ticket button and send token to activator.

5 Apply Activator's Token

1. Copy new token from activator.
2. Paste into "Paste Token Here" and click Apply.

6 Launch Game

Run NeedForSpeedUnbound.exe directly.
